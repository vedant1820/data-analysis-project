# Chocolate Sales Data Analysis


## Project Overview
This project analyzes chocolate sales data for **India** using Python and Jupyter Notebook. The dataset contains sales records from multiple countries, but this analysis focuses exclusively on the Indian market. The project includes data cleaning, exploratory data analysis (EDA) and Visualizing to identify sales trends and generate business insights.


## Tools and Technologies

- Python
- Jupyter Notebook
- numPy
- Pandas
- Matplotlib


## Project Files

- 'Chocolate_Sales.csv' - Original dataset contain Chocolate Sales.
- 'Cleaned_Chocolate_Sales.csv' - Cleaned dataset used for analysis.
- 'Chocolate_Sales.ipynb' - Jupyter Notebook with data cleaning, analysis and visualizing.
