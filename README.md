# 🍫 Chocolate Sales Data Analysis - India


## 📌 Project Overview
This project analyzes chocolate sales data for **India** using Python and Jupyter Notebook. The dataset contains sales records from multiple countries, but this analysis focuses exclusively on the Indian market. The project includes data cleaning, exploratory data analysis (EDA) and Visualizing to identify sales trends and generate business insights.


## 🛠️ Tools and Technologies

- Python
- Jupyter Notebook
- numPy
- Pandas
- Matplotlib


## 📂 Project Files

- 'Chocolate_Sales.csv' - Original dataset contain Chocolate Sales.
- 'Cleaned_Chocolate_Sales.csv' - Cleaned dataset used for analysis.
- 'Chocolate_Sales.ipynb' - Jupyter Notebook with data cleaning, analysis and visualizing.


## 📊 Analysis Performed

- Import Libraries.
- Collect Data.
- Understand the Data.
- Data Cleaning.
- Filtered the dataset to analyze **India** only.
- Exploratory Data Analysis (EDA) for India.
    - Sales Performance.
    - Monthly Analysis.
    - Product Analysis.
    - Sales Person Analysis.
    - Yearly Growth Analysis.
    - Boxes Shipped Analysis.
    - Country Filter (India's Contribution to Global Chocolate Sales).
- Saved the cleaned dataset as a CSV file.


## 🚀 How to Run

1) Make sure you have tools like VS Code or Jupyter Notebook.
2) On VS Code you have to install the python extentions.
3) On Jupyter Notebook you can directly run on them.


## 🎯 Project Objectives

The objective of this project is to analyze chocolate sales data for **India**, identify sales patterns, compare product performance, and present meaningful insights using Python and data visualization techniques.
